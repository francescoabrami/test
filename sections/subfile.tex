\section{Subfile Template}

This is a template for a subfile for a \LaTeX\ project.
To be able to use this file you need to call it in the \textbf{main.tex} file with the following command:

\begin{center}
	\textbackslash subfile\{section/subfile.tex\}
\end{center}

The path can be edited to import file from different locations.

You may replace this section between the code comment to write your own.\\

\textbf{Note:} All subfile from this template always start as a new page, if you wish not to remove the \textbackslash newpage at the top of the file.
